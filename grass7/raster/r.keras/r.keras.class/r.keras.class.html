DESCRIPTION

r.keras.class represents a front-end to the keras and tensorflow python package. The module enables keras classification models to be applied to GRASS GIS rasters that are stored as part of an imagery group group or specified as individual maps in the optional raster parameter. Several commonly used classifiers are exposed in the GUI and the choice of classifier is set using the classifier parameter.
